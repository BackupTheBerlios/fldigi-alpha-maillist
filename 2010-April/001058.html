What I do, and I'll admit this is an acquired taste, is to run the
hardware clock in UTC and in Windows declare that I am in Casablanca.
(Casablanca time is the same as Greenwich, except that Greenwich observes
DST and Casablanca does not.)

Now that I think about it this probably screws up some propagation
programs.  But since I use the Windows system almost exclusively for
ham radio it has the time the way I want it anyway.
