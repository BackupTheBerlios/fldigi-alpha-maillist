Yes, definitely. As I  have said earlier, this adjustment doesn't
generally apply to the incoming signal, but instead, corrects the
computer's error in timing. Further, no matter which (or if both), as
long as you stay on the same broadcast, the slant (and hence the
correction) will remain the same, so the unslant tuning should not be
reset between images. Let the user do that (and save it when the
program exits and restarts... mine is at .997 and every fax signal I
have seen so far is straightened by that value.
