Fldigi generates and saves a daily log to the .fldigi folder that I understand
was added to support some PC logging program.  

fldigi20101119.log
fldigi20101120.log
fldigi20101121.log
fldigi20101122.log
fldigi20101123.log
fldigi20101124.log
fldigi20101125.log

As a Mac user I have to check every few days and manually delete the files as
they keep stacking up and do not self delete after x-days.

The .log file has some value to Mac contest users as it can be used as a backup log to check contest exchanges that may have been missed or not logged correct. 

I wonder if the files could be self-deleting after x-days or if a User ID preference
could be added to disable the auto generation of the daily .log file?
