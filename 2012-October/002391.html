Dave, if you got the Olivia modem from gMFSK, that's the code I got from 
Pawel Jalocha directly and sent to Tomi OH2BNS.
gMFSK was the first and only program outside of Pawel's own test program 
to run Olivia, for quite some time.  Pawel released the code under GPL 
at my request directly.


Patrick F6CTE writes MultiPSK in Pascal, and also it is not GPL, so it 
cannot be incorporating the original code, though he may have examined 
it by some separate agreement with Pawel.  As for MixW, it's also 
proprietary and I don't know the implementation language, so either it 
was re-implemented or made available directly by Pawel.
