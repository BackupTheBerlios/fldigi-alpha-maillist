I've run into issues where with FLDIGI copying MixW stations the text is
garbled with Olivia. I would switch to MultiPSK and it was clear. In several
cases I found folks running 11125 sample rate on MixW and other programs
with native 44100 sound cards. Once they switched to the correct sample rate
for their sound card, the text was clear.

 

I'm not sure why MultiPSK manages to pull clean copy in these instances
where FLDIGI does not. BTW, I have never seen the garbled text issue when
all stations are running FLDIGI.
