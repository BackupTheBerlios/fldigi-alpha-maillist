I see a couple of interesting clues in Fred's original message:

> Subject: besoin d'aide pour installer fldigi sous linux mint-debian
> j'essay d'installer fldigi 3.21.38 depuis le dépot de kamal.

1. The reference to "mint-debian" suggests that Fred is probably
actually running Linux Mint.

2. The reference to "le dépot de kamal" suggests that Fred is trying to
use an fldigi package from my Ubuntu PPA repository, which is only
suitable for use on Ubuntu systems (not suitable for Mint or Debian) --
that's the reason for the dependency errors he sees.

Fldigi 3.21.38 is available for Debian Sid (unstable) in the main Debian
archive[1], for Ubuntu Precise in the main Ubuntu archive[2], and for
earlier Ubuntu versions in my PPA[3].

Assuming Fred is actually running Mint, he can either build his own
fldigi from source, or try to find a binary package provided by Mint
(google "fldigi mint linux").

Remi, if you would, please pass this information along to Fred,
translating as you think may be necessary.
