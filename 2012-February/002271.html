Remi,

Yes, thanks for the tip - I found the TX option!  That works fine.  
Also, the option to have WEFAX in either embedded or separate windows is 
a nice option to have.

About the "Tool tip" - I am referring to the little pop-up help messages 
that appear when you put the mouse pointer over a button or a field.  I 
just did some more testing and find that if I open Fldigi in some other 
mode (e.g., PSK31), then change to WEFAX, the tool tips flash up very 
quickly.  However, if I get out of Fldigi while it is in WEFAX mode, 
then open Fldigi again (hence, it opens in WEFAX mode), the tool tips 
come up normally.  Also, I found that if I open Fldigi in PSK31 mode 
(and observe that the tool tips come up normally), then change to WEFAX 
(and observe that the tool tips come up and clear in about a half 
second), then change back to PSK31, the tool tips are still "broken."
