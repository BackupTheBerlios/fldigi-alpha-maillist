If this is running on Vista or Win7, then it may be related to the 
crappy down conversion code in Windows.

Try setting Fldigi and the sound card's default sampling rate to 11025 
and see if the problem is resolved.  That will point to the crappy 
Windows down conversion code.

In PowerSDR when we transfer audio to digital mode programs using VAC (a 
virtual sound card "conduit"), we have to change the VAC default 
sampling rate set in Windows  from 44.1k to 48k to eliminate unwanted 
audio artifacts and improve SNR.  There is a how to article on our 
HelpDesk that explains the problem and how to change the default 
sampling rate of the sound device (card). We use Fldgi as an example. 
http://helpdesk.flexradio.com/entries/20306323-sample-rate-synchronization-between-vac-and-vista-win7-sound-devices
