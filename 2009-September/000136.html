First, thanks Ed for the -bg grey60 tip. That's exactly what I was 
looking for. Fldigi is looking beautiful now.

I'm seeing some differences between the way the UI Base and GTK+ buttons 
look and the way they look in Plastic.

Using the base or GTK+ UI the button "LED's or lights" are at a neutral 
background color, and it is easy to see if you have clicked the button 
because the LED lights up in the appropriate color. But using Plastic, 
and I tested thins on Mandriva 2008.1 and 2009.1, the button LED lights 
are always dimmed but at their corresponding color. The blue for TX id 
is especially hard to tell if it's lit or not as the on and off color 
intensities are so similar. Unlike the GTK+ UI, Plastic does not make 
the LED background a neutral color when the button is in the off condition.

I don't know if tis is inherent in the Plastic theme or not.
