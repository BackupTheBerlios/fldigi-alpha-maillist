Hello all,

I wonder if there has ever been any consideration to adding a CHIP modem 
to FLDIGI.  I occasionally check in to a Virginia net where they run the 
mode but have to go to multipsk.  Chip is actually not too bad of a mode 
and is pretty good for keyboard comms on 80m.

One other small suggestion (I think).  When going to op mode from the 
menu to change modes, could the program be set to allow the mode change 
to the last used, or most common submode, by only clicking on the mode 
type and without having to arrow over to the specific submode?  My work 
around on this was to create a macro set specifically for mode changes 
rather than making the change through the op mode menu.
