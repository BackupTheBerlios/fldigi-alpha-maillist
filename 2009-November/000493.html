Hello all,

I've been doing quite a bit of testing with RSID, primarily by leaving 
my rig on and monitoring some of the main digital call frequencies.

The timestamp feature is very useful as I can return to the shack and 
see what stations' rsid codes were detected and when.

Here's a thought regarding RSID.  Could a user selectable filter be 
added so that only selected modes are automatically switched upon 
receive of RSID signals?  For instance, many folks run RSID for PSK31, 
which I don't care to automatically tune to.  It would be great for our 
EmComm Team to have the ability to only have their NBEMS software only 
switch between Olivia and MT63.  Or maybe only detect MT63 then 
automatically switch back to Olivia as a default mode.

Just a thought.

-Dave, KB3FXI
