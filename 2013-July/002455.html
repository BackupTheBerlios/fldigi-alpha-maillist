Fldigi 3.21.72BM compiles and runs fine on Fedora Core 19 x86_64.

I have been getting a warning from fldigi-shell for several versions in the
.6x/.7x series.
The warning is "Use of uninitialized value $r in string ne at
/usr/local/bin/fldigi-shell line 161."

It can be fixed with a small change to fldigi-shell:161

161c161
<     if (defined $r) {print $OUT $r, "\n"};
---
>     print $OUT $r, "\n" if ($r ne "");
