Well, I just looked up my original reference:

http://www.qsl.net/zl1bpu/FUZZY/modes.htm

Quoting the Hell-80 paragraph:

> This format is FSK only - 1625 Hz white, 1925 Hz black and 1260 Hz for line signalling.

The "line signalling" tone is only used for the synchronous mode, which is not used in either MultiPSK or Fldigi. The spacing between the mark and space carriers is exactly 300 Hz, so MultiPSK is correct. I'm not sure when the spacing in Fldigi got changed, it was originally correct.
