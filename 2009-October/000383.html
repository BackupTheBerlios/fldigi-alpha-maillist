Well, there sure has been quite a discussion on this "issue".

If you have set to start a new modem at these frequencies in the
Configuration > Misc > Sweetspot then when you open fldigi it will
start at whatever modem was in use at your designated sweetspot.

If you de-select that option then fldigi will start at the last
frequency and modem settings.

The odd man out seems to be RTTY. But, just how many have ever operated
RTTY at something other than 170 shift and 45.5 baud ? So it is bug, to
me, no not really. So unless you have set a custom RTTY setting, it is
always going to open at the default (170/45.5).

Bottom line as far as I'm concerned, whats the issue ?

This was all discussed at length over 2 years ago and as Dave pointed
out, he coded it that way. It needs to stay that way.
