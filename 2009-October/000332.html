What is annoying to one is helpful to another. If it bugs you, just log a
call sign. 

It is good reminder to me that fldigi is working with a new newlog.adif file
rather than one you may have been using with stored calls.  I start each
month over with new newlog.adif file and while warnings can be annoying
it is good double-double check to me to be sure I have properly renamed
the previous month's QSO's.

If you don't really use fldigi to make contacts, I can see where it would
become annoying.  Just enter your own call and it will no longer bug you.
