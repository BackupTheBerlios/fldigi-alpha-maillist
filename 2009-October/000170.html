Greetings,

I was working Hellschreiber last night, and realized that it will not post
spots to PSKreporter since it is a facsimile mode and there is no decoded
text.  The only way I could manage to post spots of stations I copied was to
enable the configuration parameter to post spots when I save an entry to the
log, save the station I hear to the log, and then go back into the log and
delete the entry (since I was only spotting stations heard, not ones I was
in QSO with).

It would be nice if there were some provision for "post spot now" of the
callsign in the entry field - but I hesitate to suggest that another button
be added to the interface.  I suppose my workaround is sufficient, but it is
something to be considered for Hellschreiber.
