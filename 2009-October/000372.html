Travis,

I believe that the effect of backspacing is strictly related to the 
display. The original text is only lost when the display moves back and 
overwrites. The log file should have each character as received, 
followed by each of the backspaces, etc. If you somehow played the log 
file back you'd see the same thing you saw originally - the original 
text followed by the backspaces. In an editor it would probably look 
like this:  "original text^H^H^H^H" In other words, there would be no 
need to save the log file after each important message. It's all in there.

Bob W6CP
