Greetings,

We recently obtained trac hosting for fldigi at fedorahosted.org:

  https://fedorahosted.org/fldigi/

Trac comes with some nice features, including a bug tracker and user
editable wiki.  The bug tracker contains a single enhancement request
that I copied from the (now disabled) request tracker on BerliOS.
There is not very much content on the wiki right now but I hope that
this will be remedied in the near future -- we need a good FAQ list in
particular.

So here is a chance to have some early input on what should be in the
wiki.  Please take a look and send us any suggestions that you may have
on how to make it a useful resource for fldigi/flarq/etc. users.  And of
course feel free to change the editable pages and add material directly.
New pages can be created as well; details on the wiki front page.
