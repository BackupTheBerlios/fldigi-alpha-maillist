This is a very important alpha test version for Microsoft OS testers.  
As you know fldigi has been experiencing crashes on MS that would happen 
when the user would least expect it.  We believe that these are nearly 
all associated with a subtle difference between the way that MS and 
Posix systems handle exceptions.  Exceptions are a way that the software 
responds to error conditions.  When you install this version you should 
notice that there are two small dynamic link libraries in the program 
install folder.  Both fldigi and flarq require these dll's to be in the 
same folder as the exe.

Version 3.13AO

    Talker Socket 
        * Added Talker Client for Rx stream output.
          MS only - stream used by experimental text stream to ham
          speech being developed by KH6TY.

    Update win32 resource files  
        Add a common rc file with more resource information.

    Fix threads/exceptions bugs on mingw32
       * Enable dynamic linking for pthreads-win32
       * Add -mthreads to compiler and linker flags
       * Modify nsis installer script to include pthreadGC2.dll and 
mingwm10.dll

    Revert "Hamlib Exception Handling"
