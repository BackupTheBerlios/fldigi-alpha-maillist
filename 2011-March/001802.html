I no longer keep a released version, considering that the latest alpha
is the same.

I build from source and the first few times I start with the
--config-dir /test. If I see no problems I just use it as a normal
install.

I do a daily backup of my system, so I know that I will not lose
anything. 

Also think about how long it has been since an alpha had any real
problems.
