I have been testing the flrig alphas sporadically. Seems that there
just isn't enough time for the radios of late ;-S

I have noticed, starting with 1.1O and continuing to 1.1R, that
when flrig is used stand-alone, it is not possible to left-click
on the frequency in the VFO window to increase the operating
frequency of my radio. Right-clicking works just fine to lower
the operating frequency.

This issue does not manifest itself when using fldigi with flrig
as the rig control program - my radio will change frequencies up
and down without a problem.

I am (still) running an IC-746Pro, on Ubuntu 10.04LTS. I compile
from source code at each release and use different directories
for the rig configuration for each version of flrig that I have
compiled.

I'm going to attempt to test some more over the next few days.
