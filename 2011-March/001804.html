It may be worth noting that all kenwood radios report the currently
active VFO in the output of the IF command, and VFOs A and B are 2
separate entities. You have to query both with the FA/FB commands (and
FC if it's one of those radios with the subreceivers, like the 950) to
know the status of all of them. The IF command will tell the "main
display" frequency and the usage of A/B and split in one single command
instead. 

This is different than what elecraft did, they use A and B in fixed
positions, i.e you always transmit on A and receive on B etc, depending
on a few config parameters.
