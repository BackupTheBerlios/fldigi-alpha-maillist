In Windows 7 the only problem I have is when I set up the ptt in flrig
then fldigi can not activate the ptt. I can set up the ptt in fldigi
and turn it off in flrig and it works fine but then the ptt button in
flrig does not work. I was able to change frequency at the rig, in
flrig or in fldigi, and it showed correctly in all three spots.

In WinXP there was a problem communicating the frequency to the rig.
Changes in flrig showed in fldigi and fldigi changes showed in flrig.
Neither showed up at the rig. The same problem appeared in Ubuntu
linux. Since both of them run in as a virtual machine I will try to
find another machine to try it on to make sure it isn't a VM problem.
However I run HRD/DM780 and the DXLAB suite in the VM without a
problem.
