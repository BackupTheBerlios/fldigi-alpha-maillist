     Logbook Colors-Fonts

       * Added color / font configuration for logbook dialog.
       * Modified Table widget to provide column width tracking

     RX Text backspace

       * Correct logic error in handling UTF-8 char backspace

The Logbook change allows you to specify the font type and font size 
used in the logbook dialog.  The dialog controls will be resized to 
accommodate the font / font size.  The table widget (qso browser) 
retains the column proportions as the dialog is resized either by 
dragging horizontally, or by change the font / font size.

Back space characters to not wrap past the last end-of-line character(s).
