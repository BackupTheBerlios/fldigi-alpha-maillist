Hi All,

With help from Leigh, I have been able to take the CPU usage down from 
14% to 6% on each instance of fldigi.  The tricks are:

1)  Set to use linear interpolation instead of medium sync interpolator. 
(from 14% down to 8%)
2)  Check slow CPU. (from 8% down to 7%)
3)  On the waterfall in the main window, click on normal until it is 
paused.  (from 7% down to 6%)

This is using a 96KHz sound card.

Perhaps it would help to have all these optimizations in one area, or 
something so you can know what to do to make fldigi use less CPU.
