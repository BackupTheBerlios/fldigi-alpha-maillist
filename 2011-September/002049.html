Scheduled Macro
   * Added ability to schedule a single macro execution at a specific 
GMT date and time
     tag format <SKED:hhmm[:YYYYDDMM]>
     where the date defaults to current date.

Changed the UI format of the macro editor to allow more text to show in 
the tag pick list.

An example of the above is a macro named 'sked'

===============snip===============
<MODEM:NULL>
<SKED:1430>
<!GOFREQ:915>
<!MODEM:RTTY:170:45.45:5><!IDLE:2><FILE:/home/dave/arrl_test/bulletin.txt>
<!MODEM:NULL>
<!GOFREQ:1000><!IDLE:0.5>
<!MODEM:BPSK31><FILE:/home/dave/arrl_test/bulletin.txt>
<TX><RX>
===============snip===============

GMT right now is 1427.  If I press the macro button for the above macro 
the upper right hand timer button illuminates and reads SKED.  The Rx 
text pane is cleared and this text is inserted:

Macro 'sked' scheduled at 1430, on 20110921

Pressing ESC, any function key, or clicking on the waterfall will abort 
the scheduled macro.  Wait for the time to roll over to 1430 Zulu and 
the macro kicks off.
