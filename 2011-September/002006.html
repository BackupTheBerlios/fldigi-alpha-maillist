Version 3.22.0AL Alpha release

Log Server lookup
   * Added cty.dat search when connect to log server
     - cty.dat must be installed in the current fldigi
       files directory

Delayed Tags
   * Added queued tags
     - <!GOHOME
     - <!GOFREQ
     - <!IDLE
     - <!WAIT

Remember that the Inline macro tags

     - <DTMF - dual tones
     - <!WPM - CW words per minute
     - <!POST - CW post delay
     - <!PRE  - CW pre delay
     - <!RISE - CW rise/decay time
     - <!MODEM: - change to specified modem
     - <!GOHOME
     - <!GOFREQ
     - <!IDLE
     - <!WAIT

are not listed in the macro editor.  You select the immediate mode 
equivalent and then manually insert the ! character.
Each delayed tag shows as "<!" in the TX transmit buffer.  As it is 
executed it is displayed in the RX buffer using the full string and 
highlighted with the control coloring.
