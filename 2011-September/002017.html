The current implementation requires a <TX> in the macro for the 
<TIMER:nn> to be effective.

If you propose any changes please be sure that they are based on 
3.22.0AN or later.  Note that I have added the ability to delay 
execution of certain macro tags until the actual time of occurrence in 
the macro string.

Dave
