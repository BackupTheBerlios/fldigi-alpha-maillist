Open the CW "Prosigns" configuration tab Joe.  You can set the prosigns 
to be whatever character suites you. You are correct in that the default 
mapping is not the same as in the manual.

<BT> =
<AA> ~
<AS> <
<AR> >
<SK> %
<KN> +
<INT> &
<HM> {
<VE> }

Dave
