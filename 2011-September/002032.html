NULL modem
   * Added NULL modem, a device which does not receive but can enable 
transmit.
     - used as modem type before !QSY, !GOFREQ etc
     - used with <DTMF... to transmit DTMF tones and no other modem 
audio stream
   * Modified IDLE test in get_tx_char

Please try the following macro (suggest transmitting into a dummy load :-) )

Label it "ARRL" since it it similar to an ARRL digital mode broadcast.  
Mark and copy the text and then paste it into the macro editor text box.

<MODEM:NULL>
<!QSY:3594:915>
<!MODEM:RTTY:170:45.45:5><!IDLE:2>
ryryryryryryryryryryryry
<!MODEM:NULL><!GOFREQ:1000><!IDLE:5><!MODEM:BPSK31><!IDLE:2>
The quick red fox jumped over the lazy brown dog.
de <MYCALL> k
<TX><RX>
