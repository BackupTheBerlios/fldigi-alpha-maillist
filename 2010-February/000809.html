> What is the Server String ?

The server string specifies how to reach the pulseaudio daemon. Normally
it is left blank; the PulseAudio library knows how to contact the local
daemon.

  http://pulseaudio.org/wiki/ServerStrings

> Is there some special setting in /etc/pulse ?

No special configuration should be necessary AFAIK, but it really
depends on your distibution.
