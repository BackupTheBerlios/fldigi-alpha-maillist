> I was just checking with transmit RSID and it doesn't seem to work on

Fldigi maps FM_HELL_105 (RSID code 107) to its FSK Hell-105 mode.
FM_HELL_245 (RSID code 108) is mapped to an invalid mode, which causes
the "RSID: xxx unimplemented" message to be logged.

So it seems -245 is another FSKH variant like -105.  If so, we could add
it and then it would be supported.
