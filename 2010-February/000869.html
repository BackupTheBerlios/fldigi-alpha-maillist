Hamlib 1.2.10 has been packaged for debian by Kamal Mostafa KA6MAL and is
included in the upcoming Ubuntu 10.04 release (Lucid). He has also made a test
package available for Ubuntu 9.10 (Karmic). That package is available from the
ubuntu-hams PPA (personal Package Archive) here:

https://launchpad.net/~ubuntu-hams/+archive/ppa

I know that fldigi dropped back from hamlib 1.2.10 after some problems were
found. Is there anyone here who is familiar with whatever the problem was and
can test 1.2.10 again when we have a package available? I can't find the problem
mentioned in the archive.

I'll be testing on Karmic with a FT-767, but as I recall it was rig-specific for
a different radio.

Hams using Ubuntu linux are invited to join the Ubuntu amateur radio team:
https://launchpad.net/~ubuntu-hams
