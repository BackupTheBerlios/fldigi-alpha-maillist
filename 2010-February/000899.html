Currently a "hotlink" to the freq/mode can only be embedded in the Rx
pane with a left-button double-click in the waterfall (or single-click
if selected in the Waterfall/Mouse config). This first retunes the freq
to the current cursor position, which might not be desired.

It would be useful to be able to generate a hotlink without affecting
the current tuned frequency. As nothing is currently assigned to a
centre mouse-click this seems a natural assignment (e.g. tune with
mousewheel, click it to generate link).
