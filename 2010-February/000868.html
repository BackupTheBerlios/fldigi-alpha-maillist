In the Contest configuration "RST Always 599" is checked. Save and 
close. Are you doing this step every time ?

On the main qso window I click the clear/broom box. The In/Out boxes are 
now populated.

Under File > Logs I click Start new log.

I open the log, but keep it minimized. I check it periodically to make 
sure there are no errors that need corrected. Don't forget that you can 
make updates/changes to the log on-the-fly.

I also check the Log all RX/TX text box. To me this is a valuable post 
contest tool.

I do not use the <RST> macro tag ::

<CALL> 599  <XOUT>  <XOUT>  <CALL>  K

I do use the <LOG> macro tag. How are you logging the QSO ?

After the contest I always double check the log by generating a text 
file. I find this makes it easier to spot errors. I correct any errors 
prior to generating a Cabrillo.

If I close and restart fldigi for any reason, I always open the contest 
configuration tab, double check I have the correct log open, etc.
