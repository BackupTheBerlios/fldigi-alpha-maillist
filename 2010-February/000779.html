Hello,

I raise a small proposal we have around here in Cuba, where having a good
radio (a radio factory with a working CAT) is a problem because the
choises are not in hand as easy.

I personally have mandated to make promotion to use to fldigi to my
colleagues and we are already using it, but almost none has a radio with a
working CAT (several alphas as hamlib, but that do not work well,
FT-747GX, IC-706, IC707, etc. ).

So we want when it is no checked the use of rigcat/hamlib option (rig
control deactivated, beside the PTT via lptX/ttySX/COMX), the working
frequency (dial), radio mode (USB/LSB/CW/AM/FM etc) and working digimode
(PSK31/Olivia 16/500, etc) get saved to pref file, to this data survive
close and open of the progam to make easy the work with non CAT radios or
somehow enable the option that I can specify the frequency / radio mode /
digimode of start of the program (we prefer the first option discussed)

Such is our proposal. thanks.
