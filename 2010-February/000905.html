Hi all.
Further to the reported off air RSI issues.
I have been doing some RSID tests with a self generated TX file with
various PSK modes and it seems that with any mode below PSK125 the RSID
causes the cursor to be outside the area where AFC will pull it in.
I'm using version CA binary on Ubuntu 9.10 with pulse audio.
AFC range is currently 100 but I have tried wider and narrower.
the file was generated with the frequency lock on to try and minimise
drift.
Rx of the file was with lock off and RSID monitor whole band the effect
is seen even with the cursor started at the correct frequency.
