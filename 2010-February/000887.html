Version 0.3.4
    Alpha release

    Program configuration
        * Added text file compression toggle - a Config menu item
        * Moved UTC to Config menu item
        * Added preferences file to save configuration

The program can always read from an ICS transmission that was compressed 
and then wrapped.  The new toggle for compression is a TX selection.  It 
defaults to OFF.  That means that all ICS plain text files will be 
readable during the transmission.  If speed of transmission is critical 
then toggle the compression ON and the program will use compression if 
that results in a smaller file.  Remember that a compressed file is a 
binary file and therefore it must be converted to base64 before 
transmission.

The UTC selector was on the main dialog before.  It is now included on 
the Config menu.

The program will save and reuse the Compression, UTC and screen coordinates.
