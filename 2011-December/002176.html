> Is there a plan to create a packet interface just like soundmodem does ?
> It uses the Linux function openpty() to 
> create I think a pseudo pty usable by kernel ax25. This interface name 
> is then given to the kernel ax25 operations, like that:
>
> $ cat /etc/ax25/axports
> sm0     F4ECW   1200    255     7       SoundModem1 afsk1200
>
> Then this interface becomes usable by applications such : XFBB (aka 
> FBB) or Xastir. It is also possible with ifconfig (Ideally, one needs 
> an AMPRNET IP address: 44.x.y.z) , to run any network application 
> (ftp, telnet ...)
>
> I have no idea how this can be done on Windows and Apple OS but at 
> least on Linux this can be done.
