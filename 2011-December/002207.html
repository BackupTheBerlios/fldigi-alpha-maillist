Also, I was wondering if this is worth to always display the wefax send 
window in the main GUI window, because this is rarely used I guess. But 
it means that the logic of hiding the TX fax window should be the same 
whether we are in embedded mode (Everything in the main fldigi window) 
or not embedded mode (As it is now, two extra xefax windows: tx and rx).

This is where an extra button in the RX window, telling "Show tx/Hide 
tx" would help.

Maybe as well we could choose to display only one window at a time: 
wefax tx or wefax rx (Default rx).

On the other hand, now the tx window is also used for displaying the 
received fax by clicking on the list of received files: This is 
convenient because it is not needed to use Explorer or Konqueror to 
display received images. So it would be fine, after all, to always have 
the TX window open.

Any idea ?

Le 17.12.2011 23:27, remi.chateauneu at gmail.com a écrit :
> Hi all,
>
> This work is nearly finished, more or less following expressed opinions,
> except an ergonomic detail for which I would like your advice.
>
> Now, there is a new tab for wefax modem, where one can click whether
> wefax windows are embedded or not. By default, it is set to no.
>
> If it is set, the two windows now open at the place of the usual rx/tx
> text windows, and it works as usual.
>
> If it is not set, the tx and rx windows are blanked (because there is no
> text to display or send) and the existing receive and send windows are
> open.
>
> If the fax receive window is closed, it is still possible to reopen it
> bu going to the View/Weather fax tab, just like now.
>
> If the send window is clsoed, it is not possible to reopen it because it
> was done by right clicking in the text send window (Which is now hidden
> when in wefax, not-embedded mode).
>
> On top of that one may find annoying to have the two fax windows (rx and
> tx) open together.
>
> It is possible to adjust this behaviour by different ways which can be
> combined or not:
> * Have two (instead of one) View/wefax options: Receive and send.
> * Memorize in the configuration whether the rx and the tx fax windows
> are open when going to wefax modem.
> * Add one button for TX, in the rx window.
>
> These details are a bit boring, but this is the perfect moment to make
> things exactly as people wish.
>
> Any idea welcome, thanks in advance
>
> Remi
