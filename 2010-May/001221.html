  Greetings,

  This evening I've been trying 3.21.0ABtest and in general it seems to 
  work fine.  The WEFAX decode seems to work well.

  The only bug I found so far is the following:

  If I change modes using the OpMode menu, I find no problems going back 
  and forth between WEFAX and other modes.  However, if I am in WEFAX mode 
  and use a button I've programmed with a macro button to switch to 
  another mode, e.g. PSK31 or Olivia, then Fldigi hangs and I have to kill 
  the process.  I tried to capture an event log, but it hung so badly that 
  when I kill the Fldigi process, it also clears/kills the event log so I 
  have nothing to report.

  I'm running WinXP Pro, SP3.
