 I recompiled without pulseaudio support and after a day or so I again got
 the "pulsecore/memblock.c:188, function stat_remove()" segfault! So this
 appears to be an Ubuntu problem. I purged pulseaudio from Ubuntu 9.10 and
 that fixed the periodic crashes, however it also broke a few Ubuntu apps.
 So I installed Xubuntu, which isn't tainted with pulseaudio, instead.
 Since then fldigi has run for a couple of days without crashing. So I will
 be sticking to Xubuntu in future - it's faster and less bloated than
 Ubuntu/Gnome, and I've grown to like Xfce.

 other info I can usefully gather relatedm to this problem?
