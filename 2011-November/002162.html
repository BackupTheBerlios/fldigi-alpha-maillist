On Nov 26, 2011, at 8:08 PM, w1hkj wrote:

> All:  You can use the merge log routines to cull duplicates from an existing database.  Start fldigi and then open a "new" logbook.  Import your current log into the new one and the new log will have all duplicate records removed.  Save the new log giving it an appropriate name (not the old logbook name).  The merge routine will populate the new logbook and create two adif files containing the merged and the duplicate records (only if either/or exist).  You will receive notification of the merge results and the file names both on the Rx panel and in the event log.
> 
> 73, Dave, w1hkj
