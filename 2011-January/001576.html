Dave,

I have been testing AZ this evening, and find no problems, except one 
very minor glitch with the browser.  When I first started testing AX (or 
was it AY?) you suggested deleting fldigi.prefs to fix the flashing 
browser problem.  I did that, and all was fine.  When I set up AZ, I 
found the flashing browser problem (with maximized Fldigi window) again, 
and as before found that by deleting fldigi.prefs to let it write a 
fresh version eliminated the problem.

So, I guess the instruction when 3.21 is released should be to delete or 
rename fldigi.prefs to elminate any window/browser problems.

I'm running WinXP Pro/SP3.

It's looking great Dave!
