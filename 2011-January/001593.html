Joe,

That would require linking flrig controls with fldigi macros. Probably 
not feasible, and not all fuctions, perhaps like filter selection, are 
controllable through CAT commands anyway. Are you using any rig control 
program now? With PSK31, all the stations are found in the IF passband, 
so rig control is really not necessary. The DSP filters in fldigi are 
very sharp and steep-sided, so unless you have an AGC capture problem, a 
narrow filter is not needed, and besides, in order to use the PSK 
browser, you need a wide IF bandwidth anyway.
