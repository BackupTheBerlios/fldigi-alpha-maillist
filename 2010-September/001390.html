Disabling debug.h in the file src/combo/combo.cxx solved the problem 
with openSUSE 11.2 and 11.3. For 11.1 it was not necessary to apply the 
patch.

The results can be downloaded from
http://download.opensuse.org/repositories/home:/dl8fcl/

In this repository is also a test version of hamlib available, which is 
reflecting a very actual svn state. Fldigi is build with this version.

73, Walter DL8FCL
