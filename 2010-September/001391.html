Alpha test disk image posted for OS X.

http://www.w1hkj.com/alpha/fldigi/

Remember that this is alpha software.  Test and report problems, but 
please do not use for a contest without insuring that it performs as you 
expect.

When testing on OS X the Terminal window can be your best friend.  
Install the app to your HOME directory and then open up the Terminal window

$ cd
$ cd fldigi-3.21.0AM.app
cd Contents
cd MacOS
./fldigi --config-dir testdir

The above sequence will allow you to run 3.21.0AM completely divorced 
from your normal installation.

When you are done testing you can drag the fldigi-3.21.0AM.app folder to 
the trashcan.  Everything will be deleted including the testdir
