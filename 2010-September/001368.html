Remi,

I also noticed that WEFAX creates a log entry for each received fax which
fills the log up very quickly with invalid entries that must be removed by
hand.

This might be an issue for Dave to handle - allow multiple select in the log
editor so that groups of invalid entries can be easily removed. Also a CSV
import function might be nice so that you can export to CSV, edit the
entries and re-import to a new "clean" logbook in ADIF format. It would also
serve to ensure that there are no hidden characters in the new logbook.
