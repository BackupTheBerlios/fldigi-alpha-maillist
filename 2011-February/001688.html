The fldigi Setup pane for the LoTW option should be restricted to 
ONLY the items recognized by LoTW. LoTW will ignore all others

Check Box  =    ADIF format
---------------------------------
Call              <CALL:4>VC3E
Band            <BAND:3>40M
Mode           <MODE:2>CW
Freq             <FREQ:1>7
QSO Date    <QSO_DATE:10>2006-11-25
Time ON*    <QSO_TIME:9>21:36:00Z

* LoTW converts ADIF TIME_ON to the QSO_TIME and ignores the ADIF TIME_OFF
