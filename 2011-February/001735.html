  IC7000 auto notch
    Added set_auto_notch method

  FT100D
    Added vfo A/B control methods
    Corrections to bandwidth get / set methods
    Added get_info

  FT1000MP def bw
    Added method def_bandwidth

  IC series split
    Added split method to base IC746 series
    Corrected A/B vfo, mode, bandwidth reads

  Default BW
    Added default bandwidth method to all transceivers which support
    filter selection as a function of operating mode.

  IC7600 A/B mods
    Changed access method for A/B selection
    Added bandwidth tables and CAT for set/get bandwidth

  FT1000MP swr/alc
    Added query methods for swr and alc

  RAY 152 smeter
    Created U values to smeter based on measured values.
    Measurements by Craig, WA8DRZ

  FT1000MP power scale
    Changed power scale to square law.  Expands scale below 25 watts.

  IC7600 update
    Added control methods for mode / bandwidth

  FT1000MP Bandwidth
    Implementation of full 8.2/455 filter selections

  Button A/B
    Removed A/B button
    Changed A-B select buttons to lighted button

  ICOM select race
    * removed two_vfo capability for all IC746 derivatives
    * added IC7600 CIV codes for vfo select
    * added set/get BW, Mode for both vfo-A and vfo-B

  RAY152 - vfo
    Revert to 100 Hz resolution in CAT string

Please report the good and the bad associated with specific 
transceivers.  There have been many changes and it is easy to miss where 
one change might interact with another.  If you need to observe 
additional event logging you can start flrig from the command line with 
the "--rig_debug" switch.  You accomplish the same thing on Windows by 
adding the switch statement to the launch target (desktop icon properties).

Thanks.

Dave, W1HKJ
