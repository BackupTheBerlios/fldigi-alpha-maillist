There is a dialog box in the tqsl.app called QSO Data Range 
that lets you set the starting and/or ending QSO dates  to 
select the OSO's from the input file.  

QSOs prior to the starting date or after the ending 
date will not be signed or included in the output file.

You many leave either date (or both dates) blank
