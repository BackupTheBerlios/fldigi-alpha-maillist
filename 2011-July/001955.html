Andy,

You should be able to run Fldigi and Multipsk at the same time to 
utilize the CALL ID. You just have to make sure that the transmit audio 
stays the same for each program. I've done this with my setup which 
seems to work fine with the Flex, VAC and VCP.

I haven't run into any PTT PORT issues when starting Fldigi first and 
the CPU usage doesn't change much when running both programs. Might be 
worth a try.
