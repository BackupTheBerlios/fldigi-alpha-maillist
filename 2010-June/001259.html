Version 1.0.0bF

  Menu update
    Corrected initialization of toggle menu items

  Restore Xcvr
    Added menu item to allow enable / disable of restoring
    transceiver to it's value of frequency and mode when flrig is
    started.

  FT817 mods
    Changed FT817 code to replicate the FT857D methods

  Fluid source
    Added rigpanel.fl to the distribution source package

  Read Stream
    Added ability to intercept transceiver serial stream for
    rigs requiring that service.

  IC718 Auto Notch
    Added auto-notch control code

  ICgeneric split
    * Split ICgeneric into ICbase and specific families of
      transceivers
      - IC746
      - IC756
      - IC910
      - IC7000
      - IC7200
      - IC7600
      - IC7700

  IC703 Compression
    Activated Compression On/Off control

  IC703 / TS450S
    Added new rig classes for Icom 703 and Kenwood 450S transceivers

  IC718 additions
    Added auto_notch and compression on/off controls
    Added vox on/off control

  IC718 rf gain
    Added get_rf_gain method to IC718 class.
