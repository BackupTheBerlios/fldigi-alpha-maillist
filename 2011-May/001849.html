FWIW Folks, I'm using FLDigi on an acer netbook, Unity installed on
11.04 and NOT experiencing any loss of mouse.

However, what has been described does sound like a possible video
driver issue,  if "effects" are enabled in Ubuntu, I've seen such
behaviour if using a particular refresh rate.

This netbook, not being so so powerful in video acceleration, I don't
use effects, but I have seen effects cause the strange mouse pointer
issue.
