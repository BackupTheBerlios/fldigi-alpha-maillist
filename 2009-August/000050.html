Well, there are a couple of things that I notice that you might change:

1. Set maximum sample rate for VAC to 48,000 Hz. I don't know how VAC
works but it may reserve buffer memory for a worst-case set of
operating parameters. It would need more memory to be able to buffer a
stream at 96KHz. Since 48KHz is maximum for PowerSDR, there is no need
to tell VAC that it will have to run faster than that. I would set
sample rate values for VAC to run from 8KHz to 48KHz.

2. I also see you are running the radio at a sample rate of 192KHz.
Unless you need the panadaptor display to show you 160KHz of span, you
are better off dropping the sample rate to 96KHz. It will improve the
shape of the filters if you do. When I am trying to run really narrow
modes with optimum filter shape I even drop the sample rate down to
48KHz.

Dropping the sample rate also means that the computer does not have to
work as hard. It also means more latency (delay) for the signal to
pass through the radio but for most modes that is not a problem.

Try those things and see how it works.
