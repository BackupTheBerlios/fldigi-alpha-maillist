This is a tip for those that have or plan to work the T32C Operation from Christmas Island on 10 meter FM and want to log the QSO in your fldigi Logbook. 

When I heard the T32C Operation from Christmas Island on 10 FM Phone, I used the fldigi-3.21.18 in the "SSB" Op Mode. The frequency (29.580) and radio mode (FM) displayed correct in the fldigi QSO pane; however, the logged QSO file displayhed SSB as the mode. I manually edited the mode in the logged file from SSB to PHONE. The ARRL tqsl app will accept either FM or PHONE. Do not use PH for PHONE.

Maybe the option in the fldigi Op Mode popup for SSB should be changed to PHONE?  

73 Dick AA5VU

Ps: The T32C Online log at <http://www.t32c.com/Online_Log> shows 10m FM as a separate band slot.
