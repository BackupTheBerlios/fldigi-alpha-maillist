We used the alpha version for field day at K6GBM. It was used mostly by
middle-school kids who were running our Flex radios. We used the log server
and had very few problems. The only problem was that it seemed to forget
which log file was in use after a restart. We ended up with two log files.
Either I made a mistake (very possible) or the log server forgot the last
log file opened.

No other glitches and the students were quite happy using fldigi for both
logging and PSK31. (We didn't try any other digital modes this time out.)
