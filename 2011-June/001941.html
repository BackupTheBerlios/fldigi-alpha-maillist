I just wanted to report that we used Fldigi Alpha 3.22.0AD for Field 
Day, and it worked flawlessly for us.  We have used Fldigi the previous 
two years for FD, but were able to get more operators on digi modes this 
year, and all were very comfortable with it.  We used it on a laptop, 
WinXP, Kenwood TS-440SAT (no CAT option).  I set up some macros to get 
started, and other users had no problem "fine tuning" the macros until 
we had a really nice smooth operation.

GREAT software, and it may have yielded some more Fldigi users!
