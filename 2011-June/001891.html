There are way to many different version of Linux to give a canned answer
to that.

I personally us xbindkeys and xbindkeys-config to create my keyboard
shortcuts.

You can then use a command line mixer to adjust accordingly. 

Of course you can always use Ubuntu's Keyboard Shortcuts program to do
the same thing if using their standard windows manager.

Do a search on changing volume control from the command line on google.
