Dick,

Good job on Field Day!  Sorry we did not work each other.

I'm not sure, but I think if you UNcheck the "Mode" box in the 
CONFIGURE->CONTEST window, and have only the "Band" box checked, it will 
flag a dupe if you have worked a particular station on any mode on that 
band.  This should solve the RTTY vs. PSK31 no-flag problem you had.  
However, if you are also logging SSB stations in the same log, then you 
will get unnecessary dupe flags (since you can work a particular station 
on both SSB and digi/CW on the same band, and it's not a dupe).  Maybe 
someone with more experience can verify this.

We did not run into that problem because there was plenty of PSK31 
activity on 20/15/10m to keep us busy, so we never tried RTTY or any 
other modes, and we used different log databases for digi, CW, and SSB.
