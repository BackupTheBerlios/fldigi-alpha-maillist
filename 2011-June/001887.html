The TX level output control is a blessing! It's the final touch to allow me
to use the same settings for both voice and digital modes without
readjusting the mic gain. On my setup the RigBlaster audio control is very
touchy. Having set it to the very lowest setting that would provide output
without overdriving the mic stages it meant keeping the mic level at below 3
(8:00 position) and required me to turn it up to the 12:00 position for SSB.

This is a great fine-tuning control!

The matching Input control for RX with some gain would also be a nice touch!

Thanks,
Mitch Baum / AF2M
