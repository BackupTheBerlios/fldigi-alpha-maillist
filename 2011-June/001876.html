At your convenience please test 3.21.11 which is posted

http://www.w1hkj.com/alpha/fldigi/

Test files include Linux binary, Windows installer, OS X dmg, and source 
tarball.  If someone needs a Puppy pet I can build and post.


3.21.11 includes the following changes from 3.21.10

K3 CW-QSY
      * Additions to allow qsy to function correctly when K3 is used
        in mode A1A.
      * This might also apply to other transceivers which display the
        CW transmit signal frequency on the front panel when in mode A1A.

TX Attenuator
      * Added Tx attenuator control to Audio configuration tab
      * Added main dialog short cut keys for tx attenuator control
        alt - ==> decrease by 0.1 dB
        alt = ==> increase by 0.1 dB

LOG.ADIF export
      * Added QSO_DATE_OFF to the record

73, Dave, W1HKJ
