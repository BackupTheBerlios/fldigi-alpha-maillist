Hi all and Dave.

Recently I managed to link my old FT-747GX with fldigi using a hombrew
interface cable, of course using hamlib between them.

Besides the almost 800 msec delay, and a bug in the show/real freq at
25Hz steps, so 1407010 is really 1407000; all is working fine, because
this are radio/hardware limitation...

But I propose to Dave - an all are welcome to express yourself about it
- a new feature: annotations on the CAT freq. listing...

One image is better than 1000 words... attached is a mockup of the
concept, also a link if some email filter ban the image.

http://i.imgur.com/JmzqF.png

Ham radio and SWL listening is quasi bonded together, also there are
some useful freq. off the ham radio bands that deserves to be
annotated/commented (BBC, RNW, WWV, RNE, CB for 10m opening, beacons,
etc...)

I discovered that this list can be realy huge in a little time, so (for
my) will by very usefull to have annotations/comments to describe the
freq. and schedules/contents on it

Below some examples of annotations...

- 8502Kc Wefax: Start sending about 0000 every 6 hours, SAT image at
0400/1000/1600/2200 local time sharp
- 14095Kc PSK31: ARRL bulletings: everyday but weekends at 1800L and
2100L MFSK/BAUDOT(RTTY)/PSK31
- 15000Kc AM: WWV, every XX18 minute Prop info...
- 6165Kc AM: RNW, every Sunday 0000Z, 2000L start "La estación de la
Alegría."; 00100Z Radio Noticias... etc
- 14280Kc SSB: Latin American Ragchewers...
- others as you need...
