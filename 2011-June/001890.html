Greetings,

Several months ago I realized that if I could have some keyboard 
shortcuts to adjusting the sound-card input and output levels, it would 
greatly streamline on-air operations.  I use Windows XP Pro at my radio 
station, and a search of the 'net yielded this excellent free tool 
called "Sound Control".  I installed it and customized the keyboard 
shortcuts to my liking, and it works great.  You can download it (for 
Windows systems only) free from:

   http://www.reallyeffective.co.uk/sound-control/

Dave's addition of an output control is icing on the cake for me.  Now 
(even with the Sound Control application) my output fader is closer to 
mid-range when I set Fldigi to -12 dB, and it works better.  The Fldigi 
internal keyboard shortcut is nice too.

Perhaps there is a Linux tool similar to Sound Control that someone can 
tell everyone about.

73,
-Joe-
