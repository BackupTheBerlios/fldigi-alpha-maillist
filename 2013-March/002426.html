Dave,

I have been messing around in the CQ WPX SSB contest and ran into a logging issue using fldigi. The UI/Logging configuration panel give users the option to "Default the RST to 599." This option works great for CW and Digital modes but in a SSB contest, like the CQ WPX, the default is 59, not 599.  I have to manually change each QSO in the Logbook or do a global Find "599" and Change to "59" with text editor on the Cabrillo Report after the contest.

Is there some way to easily give users the option to set the Default as 59 or 599?

Something to think about
