Posted source, binary, win-setup, OS X dmgs, and Puppy pets at 
http://www.w1hkj.com/alpha/ for

    * fldigi - version 3.21.35
    * flrig - version 1.2.6
    * flmsg - version 1.1.12
    * flwrap - version 1.3.2
    * fllog - version 1.1.1
    * flwkey - version 1.1.2

This is a fairly major update from the files posted on the download 
page.  The changes in the source and in some cases the operation of the 
program has been dictated by the need to upgrade all applications to the 
Fast Light Tool Kit 1.3.0 stable distribution.  You can still build the 
source using Fltk-1.1.10.  If you do build from the source I recommend 
updating your system to Fltk-1.3.0 (http://www.fltk.org).

fldigi and flrig also have numerous bug fixes and support updates from 
the respectively issued 3.21.33 and 1.2.5 versions.

73, Dave, W1HKJ
