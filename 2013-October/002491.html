I have pushed the 3.21.77 commit to berlios/maint/3.21

and also posted a source tarball at w1hkj.com

http://www.w1hkj.com/download/fldigi/fldigi-3.21.77.tar.gz

Please test for build and run on your various distributions.  An RsID 
decoding problem was reported and confirmed in version 3.21.76, so it is 
necessary to post the next version.  The RsID error would not be evident 
unless you were testing fldigi <=> MultiPsk. The 3.21.76xx alpha 
versions did not exhibit this RsID error.  If you have been following 
the 3.21.76xx alpha releases you should not have any surprises with 
3.21.77.

I've tested it successfully on

Mint-13 32 bit
Mint-15 64 bit
Windows XP, 32 bit
Windows 8, 64 bit
Puppy, 32 bit
OS X, G4 and Intel

A public announcement of 3.21.77 will be made on 10/31/2013 ... and this 
is not a Halloween trick or treat .
